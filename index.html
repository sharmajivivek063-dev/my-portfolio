Learnance

Learnance is a full-stack e-learning web application that facilitates one-on-one tutoring. The backend is built on a Node.js and Express.js framework with a MongoDB database for user and teacher data management. User authentication is handled via a RESTful API, which also serves as the communication layer for connecting students and teachers. The frontend provides a user-friendly interface for browsing teacher profiles and scheduling sessions. The platform's scalable architecture is designed to handle a growing user base and expand its features, such as real-time communication tools and payment gateways.

Skillscape

The Resume Dashboard Builder is a full-stack web application designed to create dynamic, data-driven resumes. Built with React (Vite), Tailwind CSS, and Node.js (Express), it enables users to input personal, educational, and professional details through an intuitive interface. The backend integrates with MongoDB Atlas to securely store data, while interactive charts and analytics visualize career insights using Recharts. The platform also allows exporting resumes and dashboards as downloadable PDFs, providing both aesthetic and functional value. This project demonstrates expertise in MERN stack development, responsive UI design, RESTful APIs, and real time data visualization for professional portfolio management.
